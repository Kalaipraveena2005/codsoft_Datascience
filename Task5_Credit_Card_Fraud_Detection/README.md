# 💳 Credit Card Fraud Detection Using Machine Learning

## 📌 Project Description

This project uses Machine Learning to detect fraudulent credit card transactions. The model is trained on a real-world dataset containing anonymized transaction features and predicts whether a transaction is **Fraud (1)** or **Normal (0)**.

The project includes:

- Data Loading
- Data Exploration
- Data Preprocessing
- Fraud Distribution Analysis
- Model Training
- Model Evaluation
- Confusion Matrix Visualization
- Feature Importance Visualization

---

## 📂 Dataset

**Dataset Name:** `creditcard.csv`

### Features

- Time
- V1 – V28 (Anonymized features)
- Amount

### Target

- **Class**
  - 0 → Legitimate Transaction
  - 1 → Fraudulent Transaction
