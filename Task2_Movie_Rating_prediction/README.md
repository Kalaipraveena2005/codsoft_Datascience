# Movie Rating Prediction with Python

This project predicts movie ratings using machine learning.

Dataset:
IMDb Movies India Dataset

Technologies Used:

- Python
- Pandas
- Scikit-Learn
- Random Forest Regressor

Steps:

1. Data Cleaning
2. Label Encoding
3. Train-Test Split
4. Model Training
5. Rating Prediction
6. Performance Evaluation
