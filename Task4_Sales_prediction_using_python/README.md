# 📈 Sales Prediction Using Python

## 📌 Project Overview

This project predicts product sales based on advertising expenditures across different media platforms using Machine Learning.

The dataset contains advertising budgets spent on:

- 📺 TV
- 📻 Radio
- 📰 Newspaper

The objective is to predict the expected Sales using Linear Regression.

---

## 📂 Dataset

Dataset Name:

advertising.csv

Features:

- TV
- Radio
- Newspaper

Target:

- Sales

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

---

## ⚙️ Machine Learning Algorithm

- Linear Regression

---

## 📊 Model Performance

Performance Metrics:

- MAE (Mean Absolute Error): 1.2748
- MSE (Mean Squared Error): 2.9078
- RMSE (Root Mean Squared Error): 1.7052
- R² Score: 0.9059
