# 🌸 Iris Flower Classification

## 📌 Project Overview

This project classifies Iris flowers into three different species using a Machine Learning model.

The model is trained using the famous Iris dataset and predicts whether a flower belongs to:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

This project was completed as **Task 3** of the **CodSoft Data Science Internship**.
